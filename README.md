## 项目简介
这个项目是一个用于地震数据分析的系统，能够根据用户指定的经纬度和半径，查询指定年份内发生的地震、滑坡等事件。
## 技术栈
- **后端**：Spring Boot
- **数据库**：MySQL
- **API文档**：Knife4j
- **数据库操作**：Mybatis
